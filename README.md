# Flask Registro de Usuarios

Aplicación desarrollada con Flask y Jinja2.

## Funcionalidades

- Formulario de registro
- Validación de datos
- Captura de:
  - Nombre
  - Email
  - Edad
  - Carrera
  - Comentarios
- Visualización de resultados
- Listado de usuarios registrados

## Tecnologías

- Python
- Flask
- Jinja2

## Ejecución

Crear entorno virtual:

python -m venv venv

Activar entorno:

.\venv\Scripts\activate

Instalar dependencias:

pip install -r requirements.txt

Ejecutar:

python app.py

Abrir:

http://127.0.0.1:5000
